# HandmadeSpaceInvaders

Space Invaders game made from scratch (as much as possible)

## Building and running

### Linux

```
./build.sh
./linux_hspaceinvaders.elf
```

### Windows
```
.\build.bat
.\win32_hspaceinvaders.exe
```

### Controls
- `A`/`D` - move left/right
- `Space` - shoot
- `R` - restart game
- `ESC` - close game
